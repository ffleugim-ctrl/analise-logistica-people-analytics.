📊 People Analytics & Logistics: Monitor de Performance (Cajamar-SP)
📌 Contexto do Projeto
Este projeto foi desenvolvido para simular e analisar a performance operacional de uma equipe de logística reversa em um Centro de Distribuição (Cajamar/SP). O objetivo é identificar gargalos de produtividade e padrões de qualidade nos itens auditados.

🛠️ Tecnologias Utilizadas
Python: Utilizado para a geração sintética de uma base de dados com 1.500 registros, simulando cenários reais de operação.

SQL Server: Modelagem do banco de dados, criação de tabelas e consultas para extração de métricas de performance.

Power BI: Criação de dashboard interativo com foco em People Analytics e KPIs logísticos.

🚀 O que foi analisado?
Volume de Auditorias: Acompanhamento do total de bips por período.

Qualidade do Recebimento: Percentual de itens lacrados vs. avariados/itens errados.

Lead Time por Setor: Média de tempo (em segundos) que cada equipe (Expedição, Estoque, etc.) leva para processar um pacote.

Ranking de Auditores: Visão individual de produtividade para suporte e feedback (People Analytics).

📈 Resultado Final
Insight Principal: Identificamos que o setor de Expedição possui o maior tempo médio de processamento (170s+), sinalizando uma oportunidade de melhoria no fluxo de saída.
